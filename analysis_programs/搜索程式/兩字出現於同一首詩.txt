兩字出現於同一首詩.php

目的：
根據徐仁甫《杜詩注解商榷》，《奉贈韋左丞丈二十二韻》中「賦料揚雄敵，詩看子建親」
一聯，「料」與「看」同義。我們希望在杜詩中，找到另外一首詩，同樣有「料」、「看」
二字，有同樣的意義，以達到以杜證杜的目的。

步驟：
1. 先找「料」字出現在哪些詩中
2. 再找「看」字出現在哪些詩中
3. 看看在兩組詩中，有沒有交集，就是兩字同時出現
4. 再看看它們的用法是不是跟《奉贈韋左丞丈二十二韻》一樣

結果：
Array
(
    [0] => 0276 賦料揚雄敵，詩看子建親。
    [7] => 4892
    [9] => 5763
)
就是說，這兩個字同時出現在另外的兩首詩中。不過，它們的用法跟《奉贈韋左丞丈二十
二韻》不一樣。

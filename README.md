<ul>
<li>版本詩文：利用某注本中【異文、夾注】的資料，復原這個注本的白文，包括異體字、夾注等。</li>
<li>兩字出現於同一首詩：找尋包含某兩個特定的、不一定相關的字的所有詩</li>
<li>搜索與坐標：從一句詩文開始，搜索某注本中的相關注釋</li>
<li>爲注釋生成坐標：用人手來標記詩文的坐標，很容易出錯。最理想的做法是自動生成坐標。這樣生成的坐標是不會錯的。這裏要嘗試的是生成【注釋】部分的坐標。
</li>
</ul>

<h2>爲甚麽要加入各種注本？</h2>
<ul>
<li>我打算在我的資料庫中加入一些宋、清的杜詩注本</li>
<li>這些注本其實在網上都能找到</li>
<li>我的目的不是提供注本的原始資料，而是加了標記（坐標）的、有利於搜索的資料</li>
<li>這些標記（坐標），全是爲了利於電腦程式搜索而設計的</li>
<li>舉個例子：我可以寫一個簡單的程式，以列出關於杜詩中某首詩、某句詩，甚至某個詞組的注釋、大意、譯文、評論，同時並列我感興趣的注家的資料，比如同時列出《杜詩詳註》、《讀杜心解》關於某詩的資料</li>
<li>我也可以同時列出某首詩的不同版本（比如說，《全唐詩》的三個版本）</li>
<li>網上提供的絕不是這些有選擇性的、有目的性的資料</li>
</ul>

鶴注。公壯遊詩云忤下考功第。放蕩齊趙間。乃在開元二十四年後。當是其時作。元和郡縣志。㤗山一曰岱宗。在兗州乾封縣西北三十里。

齊魯：史記貨殖傳。泰山之陽則魯。其陰則齊。

造化：莊子。造化之所始。陰陽之所變。
鍾：左傳。天鍾美於是。鍾。聚也。

<h2>坐標的力量</h2>
<p>我設計的坐標系統，是這個資料庫的樞紐。我之所以能把不同版本的注釋放在一起，而不用注碼，就是因爲詩文的坐標相對於版本而言，是完全獨立的；它跟各種的版本都沒有關係。反過來，不同版本的資料，卻可以圍繞著這統一的坐標，爲它服務。這裏舉一個例子，以展示坐標的潛力。在仇兆鰲《杜詩詳註》的《自京赴奉先縣詠懷五百字》一篇中，有好些段落大意。我在每段大意之前，加了相關的詩文坐標：</p>
<pre>
【大意】
〚3-6〛前三段，從詠懷敘起。此自述生平大志。公不欲隨世立功，而必期聖賢事業。所謂意拙者，在比稷契也。甘契闊，安於意拙。常覬豁，冀成稷契。《杜臆》：人多疑自許稷契之語，不知稷契無他奇，惟此己溺己飢之念而已，伊得之而納溝爲恥，孔得之而立達與共，聖賢皆同此心。篇中憂民活國等語，已和盤托出。東坡引「舜舉十六相」、「秦時用商鞅」詩爲證，何舍近而求遠耶。
〚7-12〛此志在得君濟民。欲爲稷契，則當下救黎元，而上輔堯舜，此通節大旨。江海之士遺世，公則切於慕君而不忍忘；廊廟之臣尸位，公則根於至性而不敢欺。此作兩形，以解同學之疑。浩歌激烈，正言詠懷之故。明皇初政，幾侔貞觀，迨晚年失德，而遂生亂階。曰「生逢堯舜君」，望其改悟自新，復爲令主，惓惓忠愛之誠，與孟子望齊王同意。
〚13-18〛此自傷抱志莫伸。既不能出圖堯舜，又不得退作巢由，亦空負稷契初願矣。居廊廟者，如螻蟻擬鯨，公深恥而不屑干。遊江海者，若巢由隱身，公雖愧而不肯易。仍用雙關，以申上文之意。放歌破愁，欲藉詠懷以遣意。作長篇古詩，布勢須要寬展。此二條，各四句轉意，撫時慨己，或比或興，迭開迭闔，備極排蕩頓挫之妙。
〚19-22〛中四段，自京赴奉先，記中途所見之事。此則過驪山而有慨也。歲暮陰風，將涉仲冬矣。夜發晨過，去京止六十里也。
〚23-26〛此記驪山遊幸之迹。上四，見不恤苦寒，下四，譏恣情荒樂。塞寒空，旌旗蔽天也。崖谷滑，冰雪在地也。鬰律，溫泉氣升。摩戛，衞士眾多。君臣歡娛，不恤國事。賜浴與宴，從官邀寵也。去京止六十里也。
〚27-31〛此譏當時賜予之濫。上四敘事，下六託諷。筐篚賜予，欲其活國，今諸臣皆玩忽不知，則此物豈虛擲者乎。戰慄，當思報稱也。羅大經曰：此段所云，即「爾俸爾祿，民脂民膏」之意，士大夫誦此，亦可以悚然懼矣。
〚32-37〛此刺當時后戚之奢。前八敘事，後四託諷。朱注：衞霍皆漢內戚，以比楊國忠。神仙玉質，指貴妃諸姨。勳戚奢侈而不念民窮，其致亂蓋有由矣。分帛、金盤二條，卽指驪山宴賞。《杜臆》則概指平日，謂天寶八年帝引百官觀左藏，以國用豐衍，賞賜貴妃之家，無有限極。十載，帝爲祿山起第，窮極壯麗，既成，幄帟器皿充牣其中，雖禁中不及。祿山生日，帝及貴妃賜衣服寶器酒饌甚厚，故彤庭分帛、衞霍金盤、朱門酒肉等語，皆道其實，真詩史也。
〚38-42〛下三段，至奉先而傷己憂人，仍是詠懷本意。此憶途次倉皇情狀。上六言水勢，下四言行人。羣水西來，其洶湧如此，猶幸河梁未拆耳。攀援爭渡，為川廣不能飛越也。朱注：祿山反書至，帝雖未信，一時人情恇擾，議斷河橋，爲奔竄地，所以行李攀援而急渡也，觀「河梁幸未拆」句可見。自京赴奉先，從萬年縣渡滻水，東至昭應縣，去京六十里。又從昭應渡涇渭，北至奉先縣，去京二百四十里。驪山，在昭應東南二里，溫泉出焉。又涇渭二水，交會於昭應之北，故云：「北轅就涇渭。」其官渡改轍，在唐時亦遷徙無常，大抵在昭應之間，爲奉先便道耳。錢箋：謂官渡在萬年東南二十五里，不免倒說。朱注則指涇陽縣涇水之渡，路又隔遠。至舊註引《魏志》官渡，不切。唐之萬年，卽今咸寧。唐之昭應，卽今臨潼。唐之奉先，卽今蒲城。
〚43-48〛此述家人困窮境況。上四在途而歎，下八至家而悲。《杜臆》：敍父子夫婦之情，極其悲慘。寄跡他鄉，故秋禾呈登，而無救於貧。
〚49-52〛末以憫亂作結，身世之患深矣。天寶季年，邊帥窮兵，故民苦租稅征伐。公在事外，尙且酸辛，況窮民之失業遠戍者乎？念及此，而憂積如山，不能掇去，又回應憂黎元意。此章分十段，八句者四段，十二句者四段，十句者兩段，錯綜而自見整齊。

此自傷抱志莫伸。既不能出圖堯舜，又不得退作巢由，亦空負稷契初願矣。居廊廟者，如螻蟻擬鯨，公深恥而不屑干。遊江海者，若巢由隱身，公雖愧而不肯易。仍用雙關，以申上文之意。放歌破愁，欲藉詠懷以遣意。作長篇古詩，布勢 須要寬展。此二條，各四句轉意，撫時慨己，或比或興，迭開迭闔，備極排蕩頓挫之妙。

歲暮百草零。疾風高岡裂。
天衢陰崢嶸。客子中夜發。
霜嚴衣帶斷。指直不得結。
凌晨過驪山。御榻在嵽嵲。

中四段，自京赴奉先，記中途所見之事。此則過驪山而有慨也。歲暮陰風，將涉仲冬矣。夜發晨過，去京止六十里也。

蚩尤塞寒空。蹴踏崖谷滑。
瑤池氣鬱律。羽林相摩戞。
君臣留歡娛。樂動殷膠葛。
賜浴皆長纓。與宴非短褐。

此記驪山遊幸之迹。上四，見不恤苦寒，下四，譏恣情荒樂。塞寒空，旌旗蔽天也。崖谷滑，冰雪在地也。鬰律，溫泉氣升。摩戛，衞士眾多。君臣歡娛，不恤國事。賜浴與宴，從官邀寵也。去京止六十里也。

彤廷所分帛。本自寒女出。
鞭撻其夫家。聚斂貢城闕。
聖人筐篚恩。實欲邦國活。
臣如忽至理。君豈棄此物。
多士盈朝廷。仁者宜戰慄。

此譏當時賜予之濫。上四敘事，下六託諷。筐篚賜予，欲其活國，今諸臣皆玩忽不知，則此物豈虛擲者乎。戰慄，當思報稱也。羅大經曰：此段所云，即「爾俸爾祿，民脂民膏」之意，士大夫誦此，亦可以悚然懼矣。

況聞內金盤。盡在衞霍室。
中堂舞神仙。煙霧蒙玉質。
暖客貂鼠裘。悲管逐清瑟。
勸客駝蹄羹。霜橙壓香橘。
朱門酒肉臭。路有凍死骨。
榮枯咫尺異。惆悵難再述。

此刺當時后戚之奢。前八敘事，後四託諷。朱注：衞霍皆漢內戚，以比楊國忠。神仙玉質，指貴妃諸姨。勳戚奢侈而不念民窮，其致亂蓋有由矣。分帛、金盤二條，卽指驪山宴賞。《杜臆》則概指平日，謂天寶八年帝引百官觀左藏，以國用豐衍，賞賜貴妃之家，無有限極。十載，帝爲祿山起第，窮極壯麗，既成，幄帟器皿充牣其中，雖禁中不及。祿山生日，帝及貴妃賜衣服寶器酒饌甚厚，故彤庭分帛、衞霍金盤、朱門酒肉等語，皆道其實，真詩史也。

北轅就涇渭。官渡又改轍。
羣水從西下。極目高崒兀。
疑是崆峒來。恐觸天柱折。
河梁幸未拆。枝撐聲窸窣。
行旅相攀援。川廣不可越。

下三段，至奉先而傷己憂人，仍是詠懷本意。此憶途次倉皇情狀。上六言水勢，下四言行人。羣水西來，其洶湧如此，猶幸河梁未拆耳。攀援爭渡，為川廣不能飛越也。朱注：祿山反書至，帝雖未信，一時人情恇擾，議斷河橋，爲奔竄地，所以行李攀援而急渡也，觀「河梁幸未拆」句可見。自京赴奉先，從萬年縣渡滻水，東至昭應縣，去京六十里。又從昭應渡涇渭，北至奉先縣，去京二百四十里。驪山，在昭應東南二里，溫泉出焉。又涇渭二水，交會於昭應之北，故云：「北轅就涇渭。」其官渡改轍，在唐時亦遷徙無常，大抵在昭應之間，爲奉先便道耳。錢箋：謂官渡在萬年東南二十五里，不免倒說。朱注則指涇陽縣涇水之渡，路又隔遠。至舊註引《魏志》官渡，不切。唐之萬年，卽今咸寧。唐之昭應，卽今臨潼。唐之奉先，卽今蒲城。

老妻旣異縣。十口隔風雪。
誰能久不顧。庶往共飢渴。
入門聞號咷。幼子飢已卒。
吾寧捨一哀。里巷亦嗚咽。
所愧爲人父。無食致夭折。
豈知秋禾登。貧窶有倉卒。

此述家人困窮境況。上四在途而歎，下八至家而悲。《杜臆》：敍父子夫婦之情，極其悲慘。寄跡他鄉，故秋禾呈登，而無救於貧。

生常免租稅。名不隸征伐。
撫迹猶酸辛。平人固騷屑。
默思失業徒。因念遠戍卒。
憂端齊終南。澒洞不可掇。

末以憫亂作結，身世之患深矣。天寶季年，邊帥窮兵，故民苦租稅征伐。公在事外，尙且酸辛，況窮民之失業遠戍者乎？念及此，而憂積如山，不能掇去，又回應憂黎元意。此章分十段，八句者四段，十二句者四段，十句者兩段，錯綜而自見整齊。
</pre>
<p>如果再把異文、夾住、注釋、評論等部分放回去，那基本上就回復到原書的面貌了。當然，要生成仇兆鰲《杜詩詳註》的原書，我還需要一個像<a href="https://github.com/wingmingchan64/Dufu-Analysis/blob/main/%E3%80%8A%E5%85%A8%E5%94%90%E8%A9%A9%E3%80%8B/template.txt">template.txt</a>的文檔，把缺掉的資料補回去。</p>

<h2>杜甫詩陣列</h2>
<p>到目前爲止，我只用簡單的文字替代的方法，來生成不同的版本。比如說，像下面這一行文字：</p>
<pre>〖岱宗夫如何〗岱宗夫[音扶]如何</pre>
<p>這行文字其實是一個指令：拿「岱宗夫[音扶]如何」來替代詩文中的「岱宗夫如何」。可是，這種簡單的文字替代帶來了一個不容易解決的問題：文字替代是無條件的。如果詩文中有好幾個「岱宗夫如何」（包括詩文中的夾注），那麽每個「岱宗夫如何」都會變成「岱宗夫[音扶]如何」。在杜詩之中，就有同一句詩句，兩次出現在同一首詩之中的情況。要解決這個問題，我直接把杜詩詩文中（不包括詩題、序言等等）的每個字分別儲存在一個名爲<a href="https://github.com/wingmingchan64/Dufu-Analysis/blob/main/analysis_programs/%E6%9D%9C%E7%94%AB%E8%A9%A9%E9%99%A3%E5%88%97.php">杜甫詩陣列</a>的資料結構當中。有了這個陣列，我就可以用坐標作導向，對詩文中的任何一個字作任何獨立的修改。當然，代價是，每修改一個字，我就得提供一個指令；我還得照顧到詩文中其他的每一個字。下面是一個沒有多大實際意義的例子（我只想展示一下，我可以怎樣利用杜甫詩陣列）：</p>
<pre>
$頁 = '0003';
$替代陣列 = array(
'〚3.1.1〛'=>'【岱',
'〚3.1.2〛'=>'宗[泰山]',
'〚3.1.5〛'=>'何，',
'〚4.1.5〛'=>'秀，',
'〚5.1.5〛'=>'[同層]雲，',
'〚6.1.5〛'=>'頂，',
'〚6.2.5〛'=>"小!】\n\n右一" );
$頁陣列 = $杜甫詩陣列[ $頁 ];
杜甫詩陣列詩文替代( $頁陣列, $替代陣列 );
顯示杜甫詩陣列詩文( $詩組_詩題, $頁, $頁陣列 );
</pre>
<p>這是結果：</p>
<pre>
望嶽
【岱宗[泰山]夫如何，齊魯青未了。
造化鍾神秀，陰陽割昏曉。
盪胸生曾[同層]雲，決眥入歸鳥。
會當凌絕頂，一覽眾山小!】
&nbsp;
右一
</pre>

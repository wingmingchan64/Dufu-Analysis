# 路徑

在樹結構中，每個節點（node），包括樹根，都有一條從樹根開始到達這個節點的固定、獨特的路綫。這條路綫，就是這個節點的（樹）路徑。

在系統內，某節點的路徑是一串從樹根開始，到達該節點所必經的所有節點的名稱；每兩個名稱之間，以逗號爲界。例子：

<pre>
0003
0003,3
0003,3,1
0003,3,1,2
0003,3,1,2-4
</pre>

路徑起點必須是文檔碼（正文樹的樹根）。

每條合法的路徑，必定與一合法的完整坐標相對應。例子：

<pre>
0003         => 〚0003:〛
0003,3       => 〚0003:3〛
0003,3,1     => 〚0003:3.1〛
0003,3,1,2   => 〚0003:3.1.2〛
0003,3,1,2-4 => 〚0003:3.1.2-4〛

0013,1,5,1,2-4 => 〚0013:1:5.1.2-4〛
</pre>

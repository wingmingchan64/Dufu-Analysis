# 坐標

坐標是杜甫詩文、杜著述文字內容單元的識別碼。在默認版本中，非成員詩坐標的格式爲：`〚文檔碼:[行碼[.句碼[.字碼]]]〛`，組詩成員詩坐標的格式爲：`〚文檔碼:[首碼:[行碼[.句碼[.字碼]]]]〛`（`[]`表示可以不出現-optional-的部分）。`〚0003:〛`指默認版本《望嶽》（「岱宗夫如何」）一詩，`〚0003:1〛`則指此詩的詩題；`〚0013:1:〛`指默認版本《題張氏隱居二首》二詩，`〚0013:2:12.1.2〛`指此組詩第二首第十二行第一句第二字「子」。

在有清楚語境的情況下，例如在某文檔之中，文檔碼就是該文檔的名字，坐標中的文檔碼可以省略。有文檔碼的坐標叫完整坐標，沒有文檔碼的坐標叫非完整坐標。

在杜著述中，例如在蕭滌非主編《杜甫全集校注》中，坐標可以帶版本簡稱，如`〚蕭0003:2:〛`指蕭書關於`〚0013:2:〛`的文字內容。在指涉杜甫詩文時，盡可能用默認版本坐標，但如果要指涉某段注文，則可以用`〚蕭0003:2:12〛`（12 還是默認版本的行碼）。如果某杜著述，如下定雅弘、松原朗《杜甫全詩訳注》，有自己的詩碼，則版本坐標可以沒有首碼：`〚訳0005:〛`對應是`〚0013:2:〛`。版本坐標也可以有次版本簡稱，如`〚郭⸨臺⸩0003:〛`。

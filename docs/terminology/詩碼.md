# 詩碼

詩碼與文檔碼有密切的關係，又有重要的分別。古詩中有組詩，幾首詩編在一起，共用一個詩題，如《秦州雜詩二十首》。組詩中的成員詩可以各自有自己的編碼，如謝思煒《杜甫集校注》、下定雅弘、松原朗《杜甫全詩訳注》中的成員詩都有自己的碼。這個專屬於一首詩（不管是否組詩的成員詩）的編碼，叫詩碼。

一首詩如果不屬於任何組詩，則與這詩對應的文檔碼就是這詩的詩碼。例如，在默認版本中，`0003`旣是《望嶽》（「岱宗夫如何」）一詩的文檔碼，也是這詩的詩碼。

如果一首詩是某組詩的成員詩，除非這首詩在某杜著述中有作者編的詩碼，我一律用`〚文檔碼:首碼:〛`作成員詩的詩碼，用`文檔碼-首碼`來表示詩碼文檔名字。例如默認版本中的《題張氏隱居二首》，坐標是`〚0013:〛`，文檔名字是`0013.txt`；`〚0013:1:〛`、`〚0013:2:〛`分別是二詩的坐標，`0013-1.json`、`0013-2.json`則是裝載二詩基準正文樹的文檔。

與詩碼直接相對應的，可以是詩題（非組詩），也可以是成員詩的副題（不是詩題）。

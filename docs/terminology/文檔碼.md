# 文檔碼

實體文檔的識別符號。文檔是一個容器，可以裝載一首杜甫詩，像《北征》，包括詩題、題注、序言、詩文；也可以裝載一組詩，像《遣興五首》。文檔可以裝載討論這首詩的注文、評論，等等。文檔碼就是這個文檔的名字。與文檔碼直接相對應的是詩題；每個詩題（詩題可以重複）都有一個獨特的文檔碼。

文檔碼可以細分爲默認版本文檔碼、其他版本與著述的文檔碼。默認版本文檔碼是我自己專用的文檔碼。這套碼源於蕭滌非主編的《杜甫全集校注》，是該書的頁碼。但我另給《杜甫全集校注》獨立編碼（文檔碼、詩碼），因此兩套碼已經沒有任何關係。

文檔碼有幾種表現形式：

- 坐標： `〚0003:〛`，指《望嶽》（「岱宗夫如何」）
- 文檔名字： `0003`，如果出現在默認版本文件夾中，指默認版本的 0003《望嶽》的文檔；在蕭滌非主編《杜甫全集校注》的文件夾中，則指《題張氏隱居二首》的文檔
- `〚X0003:〛`：版本文檔碼，X 爲版本名稱簡稱，如`〚蕭0003:〛`（《題張氏隱居二首》）
- `〚郭0003:〛`、`〚郭⸨臺⸩0003:〛`分別指郭知達《新刊校定集注杜詩》中的《贈李白》（「二年客東都」）、郭知達《新刊校定集注杜詩》臺北故宮藏本的《贈李白》

# 基準正文

基準正文是某詩、某文的基準文本；此文本是生成其他版本、注本的出發點。「基準」二字，完全沒有「正確」、「標準」的含義，只說明這是整個生成文獻過程的出發點而已。

換一個角度，基準正文可被視爲眾多版本、諸多差異之間的最大公約數。生成、編輯一個基準正文文檔，必須遵從爲這基準正文設計的特定格式。以下是幾個步驟：

- 去除異體字
- 除了用「。」來斷句，去除其他所有標點符號
- 空行有特定的意義、用途；依文本需要而加入適當的空行

從基準正文出發生成基準正文樹，生成的程式必須依照文獻的需要而決定如何處理空行。例如，在處理宋詞時，詞正文中的空行代表上、下（或第一、第二、第三）闕的分界；在小說（如《紅樓夢》）中則代表段與段的分界。

# 基準正文樹 （Canonical Text Tree）

基準正文樹是詩文的一種結構化表示方式。文本被表示爲一棵層級結構的樹（如：詩 → 行 → 句 → 字），其中「字」爲最小單位。該結構可作爲文本處理與版本生成的基礎。

## 優點

### 1．單一文本來源，多版本生成

一棵基準正文樹可以生成多種不同的文本版本，例如：

- 異文版本
- 帶夾注版本
- 不同校本
- 不同排印形式

因此只需維護一個基準文本，卽可派生多種版本。

### 2．支持靈活的結構操作

由於文本以樹結構表示，可以進行多種結構性操作，包括：

- 字級操作：刪除、替換或插入單個字
- 子樹操作：插入或刪除整句、整行或整聯
- 結構調整：互換兩個非終節點（如兩句或兩行的位置）

這些操作可用於處理異文、補字、版本差異等情況。

換字、刪字：

	// 換字
	替換路徑字( $樹, 
		完整坐標轉換成路徑列陣( "〚${詩碼}:3.2.1〛" ),
		'迴' ); // '回' => '迴'
	// 刪字
	替換路徑字( $樹,

插入夾注、數字：

	插入路徑字( $樹,

	插入路徑字( $樹,

兩句互換位置：

	// 高樓鼓角悲，風起春城暮
	$temp = $樹[ $詩碼 ][ '4' ][ '1' ];
	$樹[ $詩碼 ][ '4' ][ '1' ] = $樹[ $詩碼 ][ '4' ][ '2' ];
	$樹[ $詩碼 ][ '4' ][ '2' ] = $temp;

### 其他用例

- 加入 markup、 HTML tags 等等
- 加入 emojis、 media links
- 加入外語譯文
- 加入現代標點符號（如奇句用逗號，偶句用句號）
- 更換整首詩的文字內容

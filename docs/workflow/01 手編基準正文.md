# 01 手編基準正文

## 槪述

基準正文爲整個系統的起點，由此可以生成基準正文樹、搜索用的索引等。

---

## 格式

- `.txt` 文字檔
- 第一行：作者名字、詩題、文章篇名等
- 第二行：空行
- 第三行：序言、組詩第一首的副題、正文開始
- 據語義分行（如長篇對話、宋詞押韻）
- 用空行分段（段有段碼）

例： 《望嶽》

<pre>
0003 望嶽

岱宗夫如何。齊魯青未了。
造化鍾神秀。陰陽割昏曉。
盪胸生曾雲。決眥入歸鳥。
會當凌絕頂。一覽眾山小。
</pre>

例： 《題張氏隱居二首》

<pre>

## 杜著述之段與行的運用

- 在正文樹中，段有段碼，行有行碼
- 行是段的子節點
- 行碼是該行文字在文檔中的實際行碼，方便尋找文字
- 段碼則以空行爲界，從 1 開始，方便引用
- 傳統杜著述多用句注、聯注，注文偏長；依據原書，每個獨立的注文爲一段
- 傳統杜著述用段碼
- 段中可分行，方便把大段的注文分解成意義各自獨立的幾個注釋
- 把傳統杜著述變成現代版的著述，可用數字作爲注碼，亦可分解大段的注文
- 現代版著述可用段碼+行碼

例：楊倫《杜詩鏡詮》 0001.json 開始部分

<pre>
{
    "0001": {
        "篇名": "遊龍門奉先寺",
        "1": {
            "3": "龍門卽伊闕。",
            "4": "《元和郡縣志》：伊闕山在河南府伊闕縣北，非《禹貢》之龍門。"
        },
        "2": {
            "6": "題是遊詩只寫宿。"
        },
        "3": {
            "8": "《僧輝記》：招提者，梵言拓鬭提奢，唐言四方僧物，傳筆者訛拓爲招，去鬭奢字。卽今十方住持是也。",
            "9": "《唐會要》：官賜額爲寺，私造者爲招提蘭若。"
        },
</pre>

就「招提」而言，《杜詩鏡詮》的注文爲「《僧輝記》：招提者，梵言拓鬭提奢，唐言四方僧物，傳筆者訛拓爲招，去鬭奢字。卽今十方住持是也。《唐會要》：官賜額爲寺，私造者爲招提蘭若。」，提取這段文字，可用路徑 `JINGQUAN,0001,3`。注中分別引用《僧輝記》、《唐會要》，如要分別處理（如引用原書原文與注文作比較），則可以用段碼+行碼 `JINGQUAN,0001,3,8`、`JINGQUAN,0001,3,9`。

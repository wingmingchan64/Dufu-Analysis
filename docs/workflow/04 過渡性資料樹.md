# 04 過渡性資料樹

Status: Draft

---

## 槪述

後設資料樹的路徑，作爲聯係基準正文樹、杜著述/古籍正文樹二者的指令，把杜著述/古籍正文樹的文字放到基準正文樹中合適的地方，其結果就是一棵合幷了兩種資料的資料樹。這過渡性的資料樹，就是傳送到展示單元的輸入資料，也是用以生成各種面貌的基礎。

---

## 後設資料加入的資料

在基準正文樹的原有基礎上，後設資料在相關的地方加入了以下這樣的資料：

<pre>
〈注釋*0003,5,2,1-5*仇注：曹植《冬獵篇》：張目決眥。眥，目眶也。薛夢符曰：言登覽之遠，攄決其目力入歸鳥之羣也。〉
</pre>

除了注文「仇注：曹植《冬獵篇》：張目決眥。眥，目眶也。薛夢符曰：言登覽之遠，攄決其目力入歸鳥之羣也。」外，括號 `〈〉` 內還有兩項資料： `注釋`、`0003,5,2,1-5`。這兩項資料的用途：

- `注釋` 可用作 HTML 的 `class` 値
> `<span class="注釋">仇注：曹植《冬獵篇》...</span>`
- `0003,5,2,1-5` 是條含範圍的路徑，指向「決眥入歸鳥」五字；如有需要，用者可以在這五字前後加入任何代碼，如 JavaScript 的代碼

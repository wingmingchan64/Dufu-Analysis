# From Canonical Tree to Metadata

Status: Draft

---

## Introduction

This document shows how the metadata component does the following:

- adds anchors to a canonical tree
- processes the metadata tags
- attaches metadata tags to the tree

The metadata component does not deal with rendering of the tree.

---

《古詩》：何不策高足，先據要路津。◯趙云：曹子建云：人人自謂握靈蛇之珠。呂凱《與雍闓檄》云：諸葛丞相英材挺出。&lt;/span&gt;
致君堯舜上。再使風俗淳。&lt;span class="注釋"&gt;
魏杜恕：舉明主於唐虞之上。◯增添：《孟子》：伊尹曰：豈若使是君爲堯舜之君。魏應璩《與弟君胄》書：思致君於有虞矣。◯趙云：《嵇康傳》：鍾會欲害康，曰：宜因釁除之，以淳風俗。&lt;/span&gt;
此意竟蕭條。行歌非隱淪。&lt;span class="注釋"&gt;
《前漢·朱買臣》：家貧好讀書，不治產業。艾薪樵賣給食，擔束薪，行且誦書。其妻亦負戴相隨，數止買臣毋歌謳道中，買臣益疾歌，妻羞之求去。恚怒曰：如公等，餓死溝中耳，何能富貴？買臣不能留，卽聽去。其後買臣獨行歌道中，負薪墓間。宋顏延年《詠嵇中散》詩曰：立俗遷③流議，尋山結隱淪。謝朓《敬亭詩》：隱淪旣已託。鮑照詩：孤賤長隱淪。謝靈運：旣枉隱淪客。◯趙云：鮑照《答客篇》：此意更堅滋。鮑照《發後渚》詩有蕭條背鄕心。《列子》載林類年且百歲，拾穗行歌。張湛注云：古之隱者也。舊注卻引朱買臣行歌道中負薪，此乃窮困悲歌耳，與非隱淪之義不相接。桓譚《新論》曰：天下神人五：一曰神仙，二曰隱淪。郭璞《江賦》有納隱淪之列眞。舊注引顏延年、謝朓、鮑照、謝靈運詩，皆在《新論》、《江賦》之後。此不知本始，是謂無祖者也。《世說》：周顗何如庾亮？顗曰：蕭條方外，亮不如臣。&lt;/span&gt;
騎驢三十載。旅食京華春。&lt;span class="注釋"&gt;
任昉詩：結歡三十載，生死一交情。陶潛：閑居三十載。◯趙云：《後漢·李尤》：騎驢馳村，狐兔驚走。魏文帝《與吳質書》：旅食南館。郭景純《遊仙詩》曰：京華遊俠窟。謝靈運《齋中讀書》詩曰：昔余遊京華。京華繁富之地，而當春時，尤爲繁富。於此旅食，亦不能爲樂矣。&lt;/span&gt;
朝扣富兒門。暮隨肥馬塵。&lt;span class="注釋"&gt;
鮑照詩：結交多貴門，出入富兒鄰。◯趙云：《論語》：乘肥馬。&lt;/span&gt;
殘杯與冷炙。到處潛悲辛。&lt;span class="注釋"&gt;
《顏氏家訓》：處之下座，以取殘杯冷炙。◯趙云：鮑照《野鵝賦》云：對鐘鼓之悲辛。&lt;/span&gt;
主上頃見徵。欻然欲求伸。&lt;span class="注釋"&gt;
《六韜》曰：欻然而往。《易》曰：尺矱之屈以求伸。◯趙云：《官韻》欻字注云：有所吹起貌。《神仙傳》：王母降，大茅君➃歌曰：駕我八景輿，欻然入玉清。又《莊子·庚桑楚》篇：出無本，入無竅。注云：欻然自生非有本，欻然自死非有根。又《法華經》有欻然火起之語。&lt;/span&gt;
青冥卻垂翅。蹭蹬無縱鱗。&lt;span class="注釋"&gt;
《後漢·馮異傳》：始雖垂翅回谿，終能奮翼澠池。王襃《頌》曰：沛乎若巨魚縱大壑。《海賦》：蹭蹬窮波。失勢貌。◯趙云：屈原《悲回風》云：據青冥而攄虹。玉逸《九思》曰：玄鶴兮高飛，增逝兮青冥。注：青冥，雲也。此兩句以魚鳥爲喩，一反一正，可以爲句法。◯宋玉《九辯》：悲蹭蹬而無歸。&lt;/span&gt;
甚愧丈人厚。甚知丈人眞。&lt;span class="注釋"&gt;
范元實《詩眼》曰：必言所以見韋者，於是有厚愧、眞知之句。所以眞知者爲傳誦其詩也。◯趙云：厚，言其相待之厚，蓋如《後漢》云：所以慰藉之甚厚。眞，言其懷抱之眞，蓋如《莊子》云：其爲人也眞。厚則相親愛，眞則不藏善，乃所以爲每每誦杜公佳句也。此厚與眞之義甚明。《詩眼》所謂，卻成杜公厚自慚愧於韋。杜公眞實能知韋之賢耳。非是。蓋不省厚、眞字，是詩字之足，只單著一字爲句，且用押韻，而字自有力，其義煥然也。&lt;/span&gt;
每於百僚上。猥誦佳句新。&lt;span class="注釋"&gt;
趙云：《左傳》：同官爲僚。《書》：百僚師師。誦佳句於同僚，是時公已召試賜官也。《世說》載孫興公作《天台賦》成以示范榮期。每至佳句，輒云：應是我輩語。而誦佳句三字，則隋煬帝善屬文，不欲人出其右。王胄死，帝誦其佳句云：「庭草無人隨意綠」，復能作此語耶？&lt;/span&gt;
竊效貢公喜。&lt;span class="注釋"&gt;
《前漢》：貢禹與王陽爲友，世稱王陽在位，貢禹彈冠。言其取舍同也。◯杜《補遺》：劉孝標《廣絕交論》曰：王陽登而貢公喜。子美《贈沈八丈東美除膳部員外郎》又云：徒懷貢公喜。《哭韋大夫之晉》詩亦云：貢喜音容間。&lt;/span&gt;
難甘原憲貧。&lt;span class="注釋"&gt;
《仲尼弟子傳》：原憲在草澤中。子貢相衞而結駟連騎，排藿藜，入窮閻，過謝原憲。憲攝敝衣冠見之。子貢曰：夫子豈病乎？原憲曰：吾聞之，無財謂之貧，學道而不能行謂之病。若憲，貧也，非病也。子貢慚而去。◯趙云：沈佺期《傷王學士》詩云：原憲貧無怨，顏回樂自持。&lt;/span&gt;
焉能心怏怏。&lt;span class="注釋"&gt;
趙云：《吳越春秋》：吳王僚之母謂王曰：公子光心氣怏怏，常有愧恨之色。舊注卻引《韓信》、《周亞夫傳》，乃「鞅鞅」字，又不連「心」字，非公本意所引用耳。&lt;/span&gt;
祇是走踆踆。&lt;span class="注釋"&gt;
踆踆，行走貌。張平子《西京賦》言伎戲曰：大雀踆踆。&lt;/span&gt;
今欲東入海。卽將西去秦。&lt;span class="注釋"&gt;
《語》曰：乘桴浮於海。又曰：少師陽、擊磬襄，入於海。李斯《上始皇書》：今乃卻賓客以棄諸侯，使天下之士，裹足不入秦。◯趙云：去秦，言欲捨而去耳。乃張儀惡陳軫於秦王曰：軫欲去秦而之楚。舊注卻引李斯言天下之士退而不敢西向，裹足不入秦，卻只是「不入秦」矣。&lt;/span&gt;
尙憐終南山。回首清渭濱。&lt;span class="注釋"&gt;
王粲：回首望長安。◯趙云：潘安仁《西征賦》言長安之境曰：南有玄霸素滻，北有清渭濁涇。故公凡言渭必曰清渭，言涇必曰濁涇，皆用此矣。終南山與清渭，以在秦地，故接「去秦」之下及之。&lt;/span&gt;
常擬報一飯。況懷辭大臣。&lt;span class="注釋"&gt;
《范睢傳》：一飯之德必償。《孔融傳》：一餐之惠必報。◯趙云：《李固傳》云：竊感古人一飯之報。注云：謂靈輒也。公所用主此。舊注更引《范睢傳》一飯之德必償，自是償字；又引《孔融傳》一餐之惠必報，自是餐字。以一飯之恩，嘗擬如靈輒之報宣子，況大臣相知，不獨一飯耳！其去之懷思爲如何？此詩人之情也。&lt;/span&gt;
白鷗沒&lt;span class="異文"&gt;
一作波。&lt;/span&gt;
浩蕩。萬里誰能馴。&lt;span class="注釋"&gt;
顏延年詩：鸞翮有時鎩，龍性誰能馴。謝脁詩：浩蕩別親知。東坡云：波乃沒字。宋敏求謂余鷗不善沒，改作波。殊不知鷗之滅沒煙波，最爲自然。《禽經》云：鳧善浮，鷗善沒。沒字爲是。◯趙云：何遜詩：可憐雙白鷗，朝夕水上遊。浩蕩雖本水，而不必專言水。或取流放之貌，如《離騷》云：怨靈修之浩蕩；或取曠遠之貌，如《楚辭》曰：志浩蕩而傷懷。是也。世間本多作「波」字，東坡定作「沒」字，言鷗滅沒於煙波間，而浩蕩遠去，尤有義理。而宋敏求謂鷗不解沒，作「波」字，便覺一篇神氣索然也。范淑衷甫云：世有師曠《禽經》之書，其中曰：鳧善浮，鷗善沒。則「沒」字卻是沉沒之沒，卽與前說又相反矣。&lt;/span&gt;

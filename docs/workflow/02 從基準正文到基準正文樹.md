# 02 從基準正文到基準正文樹

## 槪述

杜詩的基準正文樹生成於多年前，與 `CanonicalTextTrees` 的基準正文樹有細微的分別。此處描述的是如何生成 `CanonicalTextTrees` 中的杜著述、古籍的正文樹。

在生成正文樹的同時，系統也生成幾個搜索用的索引。

---

## 顆粒度

- 任何一個基準正文文本，都可以按需要，以不同的顆粒度生成不同的正文樹、索引
- 顆粒度指正文樹中的終端節點儲存的語言單位的大小
- 最小的單位是字；每個字可以有自己的錨（用於插入其他文字）
- 比字高一級的單位是句（以`。`爲界）；句有句錨
- 比句高一級的單位是行（以`\r\n`爲界）；行有行錨
- 比行高一級的單位是段（以空行爲界）
- 比段高一級的單位是篇（如 `0001`）；篇有篇錨
- 比篇高一級的單位是文檔；文檔有文檔錨

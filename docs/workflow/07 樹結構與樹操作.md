# 07 樹結構與樹操作

Status: Draft

---

## 槪述

本文介紹如何生成樹結構，以及如何操作樹結構，從而簡單地展示樹結構的部分功能。

---

## 生成樹結構

以基準正文樹爲例，從默認版本的基準正文開始：

<pre>
0003 望嶽

岱宗夫如何。齊魯青未了。
造化鍾神秀。陰陽割昏曉。
盪胸生曾雲。決眥入歸鳥。
會當凌絕頂。一覽眾山小。
</pre>

基本步驟：
- 把全文分解成行
- 把第一行分解成文檔碼、詩題
- 其餘各行，把行分解成句，把句分解成字
- 在分解的過程中，記錄行碼、句碼、字碼
- 最後把字儲存在相應的字碼下

## 提取子樹

利用樹結構的節點名稱，可以提取樹中的任何節點。這節點可以是某子樹的根，也可以是某字的（節點）名稱。

<pre>
$子樹 = $基準正文樹[ $文檔碼 ][ '3' ];

Array
(
    [1] => Array
        (
            [1] => 岱
            [2] => 宗
            [3] => 夫
            [4] => 如
            [5] => 何
        )
    [2] => Array
        (
            [1] => 齊
            [2] => 魯
            [3] => 青
            [4] => 未
            [5] => 了
        )
)
</pre>

## 修改資料

利用字路徑，可以修改路徑末端的資料，包括改字、刪字、增字，也可以植入子樹。

<pre>
$path = '0003,5,1,4'; // change 曾 to 層
$steps = explode( ',', $path );
traverse_replace( $基準正文樹, $steps, '層' );

            [5] => Array
                (
                    [1] => Array
                        (
                            [1] => 盪
                            [2] => 胸
                            [3] => 生
                            [4] => 層
                            [5] => 雲
                        )
</pre>

## 快速搜索

有了完整的雙向樹路徑集，就可以快速地爲字句定位（location），也可以快速地提取文字（retrieval）。

<pre>
echo $map[ '0003,3,2,2' ], $newline; // 魯
echo $reverse_map[ '小' ], $newline; // 0003,6,2,5
</pre>
